---
description: Gerencia o ciclo de vida de funcionalidades
---

1. Adicionar nova feature ao Roadmap:
   `./bin/aidev feature add "<nome_da_feature>" <sprint_num>`

2. Ver status de uma feature:
   `./bin/aidev feature status "<nome_da_feature>"`

3. Finalizar e arquivar feature:
   `./bin/aidev feature finish "<nome_da_feature>"`
